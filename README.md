
Attendance management is an important administrative task in educational institutions. Traditional attendance systems rely on paper-based methods that consume class time and are difficult to maintain. Manual attendance systems may also lead to data loss, calculation errors, and inefficient report generation.

The Smart Attendance Management System is developed as a web-based solution to automate attendance recording and management. Teachers can mark attendance digitally, while administrators can manage students and courses efficiently. The system stores all records in a MySQL database, making data retrieval fast and reliable.

This project demonstrates the practical implementation of database management systems and web technologies in solving real-life institutional problems.

Frontend Developed
The frontend is developed using:
•	HTML
•	CSS
The frontend provides:
•	Login page
•	Dashboard
•	Attendance form
•	Student list
•	Course list
•	Report page

 
Backend Development
The backend is developed using PHP.
The backend handles:
•	Login verification
•	Database operations
•	Attendance insertion
•	Report generation
